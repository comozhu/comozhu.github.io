《时间里的痴人》

发表于 2019-11-25 更新于 2020-07-13 分类于 读书笔记

之前借着亚马逊免费会员下了一些小说在Kindle里，算是借的。一个月没看完，以为会自动消失，结果应该是我没联网的样子，小说都没有消失，所以还有好几本。这就是其中一本。

形式和前面读的《余生皆假期》差不多。每一章都是一个新的视角、一个也许在过去出现过的或者与过去出现过的人有所联系的人、一段未知的故事。小说开头引了几句普鲁斯特似水流年。我回头看了一下，应该只是内容、内涵意义上的致敬。现在的小说都喜欢这么写？好久以前看的《换心》还只是两个视角来回写，现在这样无疑是更加有趣也更加困难，当然不免更加

主题是摇滚乐，这基本是一部关于摇滚与美国两代年轻人的故事。看完小说，深感摇滚与美国完全可以写一部回溯式的纪实文学（也许已经有人写了吧）。四五十年间摇滚与美国不可分离，“那时候的年轻人，他们不知道在干什么”（大概有这样一句）。流浪、跟着摇滚乐手流浪；吸、一人一根吸、夜间放荡者集会吸；zisha、gewan、病态。他们的确不知道自己在什么。是的，他们太不像过去了。

本尼最后玩了个大的。我不清楚那时候他是不是疯了。他塑造了一个从不露面的摇滚巨星，聘用水军造势，办一场盛大的个人演唱会。而演唱者却是他的发小、几乎与他同龄的老头。这是他给全城人开的玩笑吗？小说中，演出十分成功，是一场空前绝后、无与伦比的盛宴。成功了，就不好分辨他是不是疯子了。

小说最后一章名为纯净声音。整本小说，也许是对摇滚乐两个关键时点的艺术性再现。第一个草创辉煌时点必然是真的，纯净声音的时点是否存在我并不清楚。也许只是小说中的希望。

第十二章实际上我没有看。这里的小说写作形式已经超出了我的阅读享受范围，像是小女孩笔触的思维导图，一张张小图里有各种人物的语言。抱歉我只是匆匆掠过（在Kindle上也太小了）。这可能是小女孩表达的艺术再现，但我尚未能欣赏。

《余生皆假期》伊坂幸太郎
《贫穷的本质》
