聚类分析粗学 Cluster Analysis

发表于 2020-07-16 分类于 方法
  • 分类 vs 聚类（真依然很拉风-简书）
    • 已知具体有几种情况，预测到底是哪种情况 vs 根据相似性聚集、分开样本（有框无框，聚类多做一步）
  • ==无明确分类标准==，以聚类算法计算数据相似性以实现分类（根据物以类聚、特性相似）
  • Hierarchical Cluster 层次聚类
    • 创建一个有层次的嵌套聚类树，存在自下而上合并、自上而下分裂两种方法
    • Bisect K-means （没懂）
    • 合并算法: 计算每一点与所有数据点之间的距离，来计算相似性（可设定阈值，最大可接受距离 ），最近两者成组，重新计算距离，重复此过程
    • 其他算法（xavier_wu-CSDN）
  • Partitional Cluster 分隔聚类？？
    • K-means（李启方-知乎）
      • 设定分组数K，随机选择K个点作为中心点
      • 计算其他值到中心点的欧式距离（空间各点绝对距离），近的则为一组
      • 选择组的数据中心，重复上一步
      • 直到分组不再变化，结果即为聚类分析结果
      • 步骤: 数据中心-计算距离-分组-数据中心…
      • 问题: 初始质心随意选择，导致结果不确定；不适用于非球形簇的聚类
    • K-mean及其拓展算法（xavier_wu-CSDN）
  • 密度聚类（1）（2）
  • （欧几里得空间）簇（类）评价标准: ==SSE==（误差平方和），即非质心点到所在簇质心的距离，最小
  • 距离计算
    • 点-点 欧式距离
    • 点-组 取组内各点到点的距离的平均值
    • 组-组
      • single linkage 组间最近两点，易受极端值影响
      • complete linkage 组间最远两点
      • average linkage 组间各点平均距离
