滑雪技术总结

发表于 2020-01-11 更新于 2020-07-13 分类于 运动

根据请的两个教练的说法、我的记忆与实战。由于我的实战不是很好，教练的教义过了两天也有些遗忘，技术总结必然有点问题。此总结只是为了我现在回顾、以后参考。

双板
  1. 穿鞋。很重要。良好的穿鞋状态应该使小腿前侧紧靠雪鞋的鞋舌，雪鞋后侧拉扣对应束紧，由此形成一定的支撑。此时所感受到的力，就是双板滑雪中最基本的施力模式。双板雪鞋鞋舌处有厚厚的垫子，不疼。
  2. 基本姿势。胫骨前顶，身体向前倾（绝不可向后，向后会倒），重心往前
  3. 犁式。即内八，但有的人内八刹不住。新手如我会注意到，刃立不起来，没办法铲雪减速。关键是身体前倾，不可后靠。前倾加上胫骨前顶，身体重心必然靠前，犁式可以就起作用。内八起不了作用，可能是身体没有前倾或后靠了。
  4. 转弯。有效犁式验证了你的滑雪姿态。在此姿态之上，左转弯，即右脚向左施力，左脚放松不用力。右转弯相反。初学如我，在转弯时可能不够顺畅，由此会有转弯时也如八字的情况，譬如左转弯，右脚转了，左脚还在原来的方向，两板看起来不同步。我观察教练的滑行，是同步的。他称为习惯、多练。多练练应该能行。
  5. 平地滑行。假如你会滑冰、轮滑等，那么同样的动作，找到脚掌发力至整个雪板内刃、靠其滑行的感觉，就可以平地滑行了。
  6. 进阶。小回转、大回转。教练也是这么跟我说的。当然，我不会。
单板
  1. 注意。单板初学没人手拉着手教前后退坡、落叶飘，实在过难（初级道我都摔死了，雪场可能没有平坦到适合单板初学单人联系的地方），控制不住，容易摔。建议请教练。教练可以教前后刃、换刃，其他自悟自学（教练转述她的校长语）。
  2. 穿鞋、穿板。单板鞋如普通的系带鞋，系紧一点。穿板挺麻烦的，还得每次穿，更累（不如单板穿起来轻松）。两个拉扣一个脚踝位置、一个脚尖位置（我的教练倾向于扣在脚尖后一些的位置），倒下的后脚靠板扶起，后脚仅靠穿上。后刃滑行尤需后脚靠板借力，不用长时间抬起脚尖那么累。
  3. 基本姿势。身体前、屁股后，一前一后。教练说我之所以不行，就是因为滑单板时，身体的姿势还是双板的，身体、屁股全都靠前（我有点想不起来，滑双板时我屁股靠前了吗）
  4. 后刃、前刃。后刃，将小腿往靠板上靠，自然脚尖立起，也不那么累。然后右腿微屈施力、左腿平直，自然往右（说实话，转换方向我也不知道我是怎么做的，好像其实不是这么做的。反而像是左腿用力，在滑向右边时，）。全程保持后刃，如前刃铲雪，必然向前摔，很痛。宁愿向后（请租个乌龟）。落叶飘必然有个平过来的过程（即加大立刃，左右两边同等铲雪减速，即推坡）。前刃即胫骨前靠，跪姿（不用狠跪）。前刃练起来很累，一开始脚掌由于长时间弯曲会疼。落叶飘同上。初学在落叶飘的过程中，如果板头太向下，没有办法控制速度，易摔，应注意。
  5. 换刃。看起来容易，想起来容易。我觉得我行。板头向下，先一侧刃滑，然后再换一侧。但我不会。
  6. 进阶。刻画、公园等，我还不会。
